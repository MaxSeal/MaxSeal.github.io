SQL 基础（七）子查询

1. 概述

子查询，也称嵌套查询，即一个 select 语句中嵌套了另外的一个或者多个 select 语句。

子查询的思路是：把第一个 SQL 语句的查询结果，在第二个 SQL 语句中使用，这时候第一条SQL 语句的结果，在第二条 SQL 中就可以充当一个 where 条件中的一个值，或者充当一张虚拟的表。


2. 实例

例 1：查询工资比 Smith 工资高的员工信息。


例 2：查询平均工资比 41 号部门的平均工资高的部门中员工的信息。


例 3：查询平均工资比 41号部门的平均工资高的部门中员工的信息，并且显示出当前部门的平均工资。
